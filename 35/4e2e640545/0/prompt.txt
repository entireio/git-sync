3. *No ctx.Done() check inside translate* — for a million-object kernel-scale conversion, Ctrl-C during translation won't be responsive until the fetch finishes its outer phase. The signBranchTips and the git fsck step do honor ctx via exec.CommandContext. Worth adding a select { case <-ctx.Done(): return ZeroHash, ctx.Err(); default: } at the top of translate (or in the entries-loop in translateTree) for big repos.
