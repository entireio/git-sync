4. *HEAD repoint condition* (sha256convert.go:472-479) — only repoints if refService.HeadTarget is set and present in desired. If a server doesn't advertise HEAD (some v1 servers), HEAD remains the PlainInit default refs/heads/master, which then fails the --check HEAD step on a main-default repo. Minor — but the --check failure would be confusing. Consider falling back to refs/heads/main or the first available branch.
